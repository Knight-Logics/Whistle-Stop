Staff: Update schedule

No Wix. No Canva every month. Edit one file — the whole site updates.

  1. Open site/data/events.json in any text editor (or Notepad).
  2. To add a Friday band, add an object to "performances": 
    {
  "date": "2026-06-13",
  "title": "Artist Name",
  "startTime": "18:30",
  "endTime": "21:30",
  "category": "live-music"
}
  3. Recurring nights (tacos, cornhole, book club) live in "recurring" — change once, applies every week.
  4. Save the file and refresh the website. Calendar + upcoming lists update automatically.
  5. Google Business Profile: optional — connect a Google Calendar to auto-sync GBP events so you're not typing Dave Sansoucie twice.
